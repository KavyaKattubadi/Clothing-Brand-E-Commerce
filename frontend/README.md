# Frontend - Minimal React App

This is a minimal React scaffold with basic pages to interact with the backend.
You can create the frontend using Create React App or Vite and copy the `src/` folder.

Key files provided in `src/`:
- App.jsx
- index.jsx
- pages/Home.jsx
- pages/Products.jsx
- pages/ProductDetail.jsx
- pages/Login.jsx
- pages/Register.jsx
- pages/Cart.jsx
- pages/Checkout.jsx

The frontend expects the backend API at http://localhost:5000 (update base URL as needed).
